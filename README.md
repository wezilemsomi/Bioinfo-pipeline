#Bioinformatics Pipeline

##Project Description 
This pipepline analyzes sequencing data. It includes basic steps such as input validation, quality control filtering, and structured output generation for downstream analysis.

The aim of this pipeline is to improve reproducibility and ensure consistent handling of sequencing datasets.

##Dependencies 
The following tools/ libraries are required: 
Python 3.10
Pandas
Numpy 

##Installation 
1. Clone the repository 
example: 
git clone https://github.com/wezilemsomi/Bioinfo-pipeline.git
cd Bioinfo-pipeline

2. Install required packages
pip install biopython pandas numpy

##Execution 
run this pipeline using 
python main.py --input data/sample.fastq --output results/
